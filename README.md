# Task Management Dashboard

A high-fidelity, premium task management dashboard built with React and Tailwind CSS. This project focuses on a clean, human-readable UI with custom-built components ranging from advanced data visualization to real-time interactive widgets.

## Key Features

- **Dynamic Navigation:** Minimalist sidebar with a toggle system for expanded or compact views.
- **Project Analytics:** Custom-built bar charts using Recharts, featuring diagonal stripe patterns for "pending" states and smooth pill-shaped bars.
- **Active Tracking:** Real-time Time Tracker widget with a premium dark-green aesthetic and background pattern effects.
- **Team Collaboration:** Dedicated section for monitoring team task progress with custom status labeling.
- **Data Visualization:** Semi-circular project progress gauge for quick completion rate analysis.
- **Responsive Layout:** A modular grid system that keeps widgets balanced and leveled across different screen sizes.

## Tech Stack

- **Frontend:** React 18
- **Styling:** Tailwind CSS (Utility-first approach)
- **Icons:** Lucide React
- **Charts:** Recharts
- **State Management:** React Context API (AuthContext)
- **API Handling:** Axios (with custom secure hooks)

## Getting Started

Follow these steps to get the environment running locally:

### 1. Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/rakib-hossain32/task-management-dashboard.git
cd task-management-dashboard
npm install
```

### 2. Environment Setup
Create a `.env` file in the root directory (if required by your backend) and add your local or production API URLs.

### 3. Run Development Server
```bash
npm run dev
```

## UI & Design Principles

The design follows a "Modern Editorial" aesthetic:
- **Spacing:** Generous white space and rounded corners (32px) for a premium feel.
- **Color Palette:** Tailored deep greens (`#0D3D29`), light mints, and soft grays.
- **Typography:** Clean, sans-serif fonts to ensure readability even with complex data sets.
